# Bang & Olufsen Landing Page

A modern and responsive landing page for **Bang & Olufsen**, designed to showcase premium audio products, featured collections, and company information with a clean, elegant, and user-friendly interface.

## Project Description

The **Bang & Olufsen Landing Page** is a single-page website inspired by the premium design aesthetic of Bang & Olufsen. It presents the brand's products, highlights featured collections, provides company information, and includes contact details. The layout is fully responsive and optimized for different screen sizes, delivering a seamless user experience across desktop, tablet, and mobile devices.

## Technical Requirements

To run this project locally, you will need:

* **Node.js** (version 18.x or newer) – JavaScript runtime environment.
* **NPM** (version 9.x or newer) – Package manager for installing dependencies.

## Installation and Setup

Follow these steps to run the project locally:

### 1. Clone the repository

```bash
git clone https://github.com/Wolfymmm/Bang-Olufsen-landing-page.git
```

### 2. Navigate to the project directory

```bash
cd Bang-Olufsen-landing-page
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm start
```

## Usage

After starting the development server, the application will be available at:

```
http://localhost:8080
```

You can preview the landing page in your browser, edit the source files, and see changes instantly during development.
